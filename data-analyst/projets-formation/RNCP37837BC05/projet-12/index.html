Projet 12 - Détectez des faux billets avec R ou Python

Bloc RNCP : RNCP37837BC05 - Spécialisation Data Analyst - Statistiques : Réaliser des tests statistiques et concevoir des modèles d’apprentissage supervisé et non supervisé

Durée indicative : 70 heures

---

Compétences du projet

  • Utiliser un modèle d’apprentissage supervisé pour réaliser une prédiction

  • Entraîner un modèle d’apprentissage non supervisé pour exploiter les données

  • Évaluer et comparer les performances de différents modèles

Soft skills travaillés pendant le projet

  • Communication : soutenance

  • Organistaion & autonomie : projet en autodidacte, une session de mentorat par semaine

  • Esprit analytique & critique : analyse des données et interprétation des résultats

  • Rigueur : comparaison de différentes méthodes

Outils utilisés

  • Python (Pandas, Matplotlib, Seaborn, Scipy, Scikit-learn)

  • Suite Microsoft (PowerPoint)

---

Livrables du projet

  • Notebook Jupyter

  • Modèle entraîné

  • Support de présentation

  • Script Python pour prédiction
