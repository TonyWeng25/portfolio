# Projets OpenClassrooms - Data Analyst

Bienvenue dans le répertoire des projets de formation pour le parcours Data Analyst d'OpenClassrooms. Vous trouverez ici une collection organisée de projets réalisés dans le cadre de ma formation, chacun visant à développer des compétences spécifiques en analyse de données.
Chaque projet est structuré autour d'un bloc RNCP (Répertoire National des Certifications Professionnelles) correspondant, avec des descriptions détaillées des compétences visées, des outils utilisés et des livrables produits.

## Structure des projets
- **RNCP Blocks** : Chaque projet est associé à un bloc RNCP spécifique, détaillant les compétences professionnelles visées.
- **Projets individuels** : Chaque projet est documenté dans son propre répertoire, avec des informations sur les objectifs, les compétences développées, les outils utilisés et les livrables.
